# Trust Chain Starter Kit

This starter kit is a three-file static template for publishing a founder-led authority page.

If you want to distribute it directly, zip this folder or use the packaged `trust-chain-starter.zip` artifact from the main repo.

## Files

- `index.html`: the page shell
- `config.js`: editable identity and messaging fields
- `README.md`: setup notes

## First edits

1. Replace the company and founder values in `config.js`
2. Open `index.html` in a browser to confirm the config loaded correctly
3. Update any deeper method, proof, or FAQ copy in `index.html` if the defaults are not enough
4. Add real citations before publishing
